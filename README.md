# Creative-Audio-Generator
A lightweight, fully local procedural music and synthetic vocal generator built with NumPy and Gradio. Creates ambient tunes and melodic vocals offline with no external APIs. Includes SQLite logging, ADSR envelopes, formant simulation, and placeholders for advanced AI models.
